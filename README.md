# ミーティング調整アプリ

このアプリは、会議やイベントの「いつ、どの時間に集まるか」をみんなで簡単に決められるようにするためのものです。例えば、「来週中にチームで集まって、30分の打ち合わせをしたいけど、みんなの都合の良い時間を見つけるのが大変…」という時に役立ちます。

## アプリが解決する課題

*   **面倒なやり取りを減らす**: メールやチャットで何度も「この時間はどうですか？」「あの時間はどうですか？」とやり取りする手間をなくします。
*   **スムーズな日程調整**: みんなの空いている時間を一目でわかるようにして、最適な時間を見つけやすくします。


## このアプリでできること

### 1. 主催者がミーティングを設定する

ミーティングを企画する人が、最初に以下の情報をアプリに登録します。

*   **ミーティングのタイトル**: 例えば、「新製品企画会議」や「チームランチ」など、どんな集まりなのかが分かる名前です。
*   **会議にかかる時間**: ミーティングがどれくらいの時間で終わるか（例：30分、90分など）を設定します。
*   **候補日と時間帯**: 「この日に集まりたい」という日（例：2月5日、2月11日）をいくつか選び、さらにその日の中で「この時間帯ならミーティングができる」という範囲（例：20:00から23:00の間）を設定します。

これらの情報を登録すると、ミーティングのイベントが作成されます。

### 2. 参加者が自分の空き時間を登録する

イベントへの招待を受け取った参加者は、自分の名前を入力し、主催者が提示した候補日と時間帯の中から、自分が「この時間なら参加できる」という空いている時間を登録します。
例えば、2月5日の20:00から23:00の候補時間帯に対して、「私は20:30から22:00までなら空いています」というように登録します。

### 3. みんなで参加できる時間を確認する

主催者も参加者も、誰がどの時間帯に空いているのか、そして**参加者全員が共通して参加できる時間枠**がどこにあるのかを、アプリの画面上で一目で確認できます。これにより、最適なミーティング時間を簡単に見つけることができます。
